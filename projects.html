Internship Projects

1. Static Content Promo

During my internship period at Alfagift - PT Global Loyalty Indonesia, I was given the responsibility to handle the campaign promo that are available within the Alfagift mobile application by using HTML and Bootstrap framework as the main programming language. In addition, several tools are also utilized to support this task including Microsoft Visual Studio Code, Jira Software, Alfacart Back Office, Google Cloud, Jenkins, and others.

2. Alfagift Design System Web

Alfagift Design System web is a web documentation that is used to store static promo campaign content modules. In this project, I was assigned to fill in the content of each menu from the web framework that had been developed by my mentor. To be more specific, I created code snippets and mobile mockups that can be run as prototypes using React JS.

3. Alfagift Affiliate

Alfagift Affiliate is the latest feature that has just been developed in the Alfagift application. Alfagift Affiliate Program is a promotional program for Affiliate Products promoted by Alfagift Affiliators through social media using the promotional links provided. In this project, I was given the task by my mentor to convert the UI design that my mentor had made into code using the Nuxt.js framework.

Coded

In this Affiliate project, I was given the opportunity to create code for the dashboard, profile, and info-affiliate sections. This code is then used as a layout for the FrontEnd side of this feature and injected into the Alfagift mobile application using a webview.

4. HTML Body Email

I also contribute in creating HTML code for email body templates that will be used and sent to Alfagift users about 1 to 2x every month. This email is usually a voucher discount notification email or an Alfagift member predicate notification email.

5. APO Dashboard

APO stands for Online Order Application. This application aims to assist Alfamart store employees in receiving, preparing, and tracking the delivery of online orders. In this project, I was assigned to create this static design page using HTML CSS.

College Projects

1. LifeLine

Lifeline is an app that aims to assist users in booking an appointment with a psychologist. Users can choose the psychologist they want to see and also determine the schedule of their meeting with the psychologist. In addition, the LifeLine app also provides a psychotest feature that each user can take and users can view the test results they have done.

2. Wild Life Hunting

Wild Life Hunting is a virtual reality game themed on Indonesian fauna designed to be played on VR devices. This game focuses on the concept of hunting animals where players are tasked with hunting animals such as deer with a first person shooter perspective. This game consists of several levels or missions that must be completed by the player in order to get the achievement.

3. LibSpot

LibSpot is a web-based application that acts as a digital library that provides electronic academic documents to students legally. Through this web, students are allowed to search for a document, view detailed information from a document, download documents, make reviews, and make reports regarding bad reviews or content issues on documents. This web is designed to have a user-friendly appearance to suit all ages of users and has a flow that is easy to understand.
